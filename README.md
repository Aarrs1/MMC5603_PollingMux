## 简介：ESP32-S3 控制双 I2C 多路复用器与 16 路 MMC5603 传感器测量系统

该 Arduino 程序基于 **ESP32-S3**，通过两个 **I2C 多路复用器（地址 0x70 和 0x73）** 控制 **16 路 MMC5603 三轴磁力计传感器**，实现高频、非阻塞的并行测量与数据采集。

### 核心功能

1. **I2C 多路复用器控制**

   * `selectChannel()` 函数可快速选择指定通道，通过控制字节启用单通道，支持双复用器共 16 路传感器操作。
   * 延迟微秒级切换（<1µs）保证高速数据采集稳定性。

2. **MMC5603 初始化与测量**

   * 所有传感器在启动时自动启用 **SET/RESET 去磁**，带宽设置为最大值（测量时间约 1.2ms）。
   * 支持单次测量启动（`startMeasurement()`）及多字节寄存器读取（`readMulti()`）。

3. **非阻塞状态机设计**

   * 三阶段循环：

     1. `START_MEASURE`：一次性启动全部 16 个传感器测量。
     2. `WAIT`：基于时间戳的非阻塞等待，确保测量完成。
     3. `READ`：读取全部传感器数据，并以 CSV 格式通过串口输出。
   * 避免阻塞延迟，提高系统响应能力和数据采集频率。

4. **数据处理**

   * 每个传感器的三轴磁场数据为 20 位值，通过低层位拼接完成。
   * 输出格式为一行完整帧数据，便于后端程序或 Qt 等软件实时处理。

5. **高速通信与优化**

   * I2C 设置 400 kHz，串口输出 921600 bps，减少通信瓶颈。
   * 避免 `printf` 解析开销，使用 `Serial.print` 高效输出。

---

### 使用场景

* 多点磁场成像与检测
* 并行高频数据采集实验
* MCU 数据采集与处理示例
